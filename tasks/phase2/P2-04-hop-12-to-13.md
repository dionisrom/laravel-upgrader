## Objective

Build `upgrader:hop-12-to-13` Docker image, curate L12→L13 breaking changes, implement custom Rector rules. This is the final Laravel hop. Notable: L13 requires PHP 8.3 minimum, so the container must enforce/verify this prerequisite.

---

## Context from PRD & TRD

### Key L12→L13 Breaking Changes (PRD §3.2)

- PHP 8.3 minimum requirement
- Eloquent changes
- Deprecated API removals from prior versions

### Docker Image (PRD §3.1, TRD-P2HOP-001)

| Image | PHP Base | Key Custom Modules |
|---|---|---|
| `upgrader:hop-12-to-13` | `php:8.3-cli-alpine` | PHP version guard, deprecated removals |

### PHP Version Guard (TRD §22 preview)

The hop container should emit a warning/error if the source project's `composer.json` declares a PHP version constraint below 8.3. This feeds into Phase 3's 2D HopPlanner which will interleave PHP version upgrades.
