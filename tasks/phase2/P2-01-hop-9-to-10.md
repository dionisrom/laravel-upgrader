## Objective

## Context from PRD & TRD

### Key L9→L10 Breaking Changes (PRD §3.2)

- **PHP 8.1 minimum requirement**
- **Native return types added across all framework classes** (high volume — Rector handles most)
- `Model::unguard()` removed
- Various deprecated helpers removed

### Docker Image (PRD §3.1, TRD §17.1 — TRD-P2HOP-001)

| Image | PHP Base | Key Custom Modules |
|---|---|---|
| `upgrader:hop-9-to-10` | `php:8.1-cli-alpine` | `ReturnTypeRector`, `PackageRuleActivator` |

### Breaking Changes JSON (TRD-P2HOP-002)

Each hop image bundles its own `docs/breaking-changes.json` with all changes curated for that hop.

### PackageRuleActivator Skeleton (TRD §18.1 — TRD-P2PKG-001)

```php
class PackageRuleActivator {
    public function activate(string $composerLockPath, string $hop): array;
}
```

Reads `composer.lock` → detects installed packages → activates relevant rule sets → adds to `RectorConfigBuilder`.
